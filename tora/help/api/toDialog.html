Detailed Description

A baseclass to be used for modal dialogs in TOra, basically allows F1 to pop up context help.

void  displayHelp (void)

displayHelp

[slot]

Display help for this dialog. Make sure you also inherit toHelpContext to specify the correct context for this dialog. You could use this to also have a help button in the dialog.

See also: toHelpContext
