Current Session

This tool provides information about the current session.

The interface for this tool is very simple. There are four tabs in the window. The first one present the priviledges currently granted to the user. Roles can be expanded to display what priviledges are availble through those groups. The second one present the version of the different components in the connection. The fourth and last tab display the statistics of the session. The parameters tab displays the parameters of the current session. The statistics is shown in two columns, the first one contain the actual value, the second show the change in the value since the last update. To update press the refresh button in the toolbar.
