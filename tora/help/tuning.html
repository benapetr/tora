Server Tuning

This tool provides easy access to important server indicators as well as server statistics.

The interface for this tool is very simple. There are three tabs in the window. The first one present performance indicators. The second one present the statistics available for the server. The statistics is shown in two columns, the first one contain the actual value, the second show the change in the value since the last update. The last tab displays the parameters set in the database. To update press the refresh button in the toolbar.
