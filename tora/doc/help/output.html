SQL Output

This tool display the output from the DBMS_OUTPUT package (For more information see the Oracle PL/SQL Supplied Packages manual).

Use this if you are running PL/SQL from the worksheet and want to check the output. It is also used in the debugger but as a pane in the debugger information.

The toolbar contain the following controls.

Update Poll for more output immediately.
Enable Enable or disable output collecting. Collecting is enabled if light bulb is lit.
Clear output Clear the currently collected output.
Refresh The time-out to update automatically. The default value of this is configurable though the options.
