4 Available tools

These are the available tools in a standard TOra installation. Individual installations may vary.

  • SQL Worksheet
  • Schema Browser
  • PL/SQL Editor
  • PL/SQL Debugger
  • PL/SQL Unit Tester
  • Server Tuning
  • Security Manager
  • Storage Manager
  • Sessions
  • Rollback Segments
  • SGA Trace
  • Current Session
  • DB Extraction/Compare/Search
  • Statistics Manager
  • Alert Tool
  • Invalid Objects
  • SQL Output Viewer
  • Template Help
  • SQL Editor
